Prikly

Prikly, anteriormente llamada Viuti, es una plataforma de gestión completa e intuitiva diseñada para salones de belleza, barberías, spa, fitness y más. El lenguaje que se utilizo para el backend fue C#, bajo el marco de trabajo .NET, utilizando entity framework y Dapper para la conexión a la base de datos, mientras que en el front se utilizo JS con React. El proyecto utiliza una arquitectura de microservicios y eventos de integración para construir un sistema distribuido y modular que puede escalar y adaptarse fácilmente a medida que evolucionan los requisitos y las necesidades del negocio.

© Junio/2023 ~ Tablero de ventas - versión 2.0

Visita el sitio en producción

Tecnologías: C#, .NET, Entity Framework, Dapper, React, JS, HTML, CSS, X-Unit.

Puesto: Desarrollador Full Stack
